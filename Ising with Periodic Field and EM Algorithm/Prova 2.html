(a) Solve the widow's problem: Thisted considers historic data on widows and their dependent children from aSwedishpensionfund.If $y_k$ denotes the number of widows with $k$ children,then the data values are $y0=3062$,$y_1=587$,$y_2=284$,$y_3=103$,$y_4=33$,and $y_5=4$,and $y_6=2$.

The fact that most widows have no dependent children suggests that a simple Poisson model would give a poor fit.A better model is a mixture of a population of widows with no children, population $A$,and a population of widows having a Poisson number of children, population $B$.

Suppose a widow falls into population $A$ with probability $p$ and into population $B$ with probability $1−p$.

Let $\mu$ be the mean of the Poisson distribution characterizing population $B$.

R: Seja $Y$ a variável aleatória que representa o número de filhos, então sabe-se que,

$$P(Y=0) = p + (1-p) e^{-\mu}$$

E também que, $$P(Y = y) = (1-p) \frac{\mu^ye^{-\mu}}{y!}, \text{ com } y>0.$$

Portanto, a função de verossimilhança pode ser escrita como

$$L(Y|\theta) = \prod_{y=0}^6 g(y|\theta)=g(y=0|\theta)^{y_0}\prod_{k=1}^6g(y=k|\theta)^{y_k}$$$$L(Y|\theta)= \left(p + (1-p)e^{-\mu}\right)^{y_0} \prod_{k=1}^6 \left((1-p)\frac{\mu^{k}e^{-\mu}}{k!}\right)^{y_k}$$

E tirando o logaritmo da função de verossimilhança, acabamos com,

$$\log(Y|\theta)= y_0 \log \left(p + (1-p)e^{-\mu}\right) +\sum_{k=1}^6 \left(y_k \left(\log(1-p) + k\log \mu - \mu -\log(k!) \right)\right)$$

def log_likelihood(Y, mu, p):
    a = Y[0]*np.log(p + (1-p)*np.exp(-mu))
    b = 0
    for k in range(1, len(Y)):
        b += Y[k]*(np.log(1-p) + k*np.log(mu) - mu - np.log(np.math.factorial(k)))
    return a + b

Para utilizar o algoritmo de Newton-Raphson, define-se,

$$f(\theta) = y_0 \log \left(p + (1-p)e^{-\mu}\right) +\sum_{k=1}^6 \left[y_k \left(\log(1-p) + k\log \mu - \mu -\log(k!) \right)\right] $$

E as iterações serão dadas por, $$\theta_{t+1} = \theta_{t} - H^{-1}\left[f\right](\theta_t) \nabla f(\theta_t), \text{ onde } \theta_t = (p_t, \mu_t)$$

Tal que,

$$\nabla f(\theta_t) = [\partial f/\partial p, \partial f/ \partial \mu]^T = \left[y_0 \frac{e^{\mu_t}-1}{1+p_t(e^{\mu_t} - 1)} + \sum_{k=1}^6 y_k \frac{1}{p_t-1}, \frac{y_0(p_t-1)}{1+p_t(e^{\mu_t} - 1)} + \sum_{k=1}^6 y_k \left(\frac{k}{\mu_t}-1 \right) \right]^T$$

E também temos que,

$$ \frac{\partial ^2 f} {\partial p \partial \mu} = \frac{y_0 e^\mu}{\left(p(e^\mu -1) +1\right)^2}, \qquad \frac{\partial ^2 f}{\partial p ^2} = \frac{y_0 (e^\mu -1)^2}{\left( p (e^\mu -1) +1\right)^2} - \sum_{k=1}^6 \frac{y_k}{(1-p)^2}, \qquad \frac{\partial ^2 f}{\partial \mu ^2} = \frac{y_0 p (1-p) e^\mu }{(p(e^\mu -1)+1)^2} - \sum_{k=1}^6 \frac{ky_k}{\mu^2} $$

E portanto a hessiana é dada por,

$$H[f](\theta_t)= \begin{bmatrix} \frac{\partial ^2 f(\theta_t)}{\partial p ^2} \ \frac{\partial ^2 f(\theta_t)}{\partial p \partial \mu} \\ \frac{\partial ^2 f(\theta_t)}{\partial \mu \partial p} \ \frac{\partial ^2 f(\theta_t)}{\partial \mu ^2} \\ \end{bmatrix} $$

def hessian(Y, p, mu):
    d2f_dp2 =  Y[0]*(np.exp(mu)-1)**2 / ((p*(np.exp(mu)-1)+1)**2) - sum(Y[1:])/((1-p)**2)
    d2f_dpdu = Y[0]*np.exp(mu)/(p*(np.exp(mu)-1)+1)**2
    d2f_du2 = Y[0]*p*(1-p)*np.exp(mu)/((p*(np.exp(mu)-1)+1)**2 )- sum([Y[k]*k/(mu**2) for k in range(1,6)]) 
    H = [
        [d2f_dp2, d2f_dpdu],
        [d2f_dpdu, d2f_du2]
    ]
    return H

def gradient(Y, p, mu):
    grad = [
        Y[0]*(np.exp(mu)-1)/(1+p*(np.exp(mu)-1)) + (1/(p-1))*sum(Y[1:]),
        Y[0]*(p-1)/(1+p*(np.exp(mu)-1)) + sum([Y[k]*(k/mu - 1) for k in range(1,6)])
    ]
    return grad

(b) Os dados completos, agora com a separação das viúvas $y_0$ em duas populações $x_A$ e $x_B$, são dados por,

$$ L^C(Y|\theta)= p^{x_A} + \left[ (1-p)e^{\mu}\right]^{x_B} + \prod_{k=1}^{6} \left[ (1-p) \frac{\mu^k e^{-\mu}}{k!} \right]^{y_k} $$

E portanto, o log da verossimilhança dos dados completos é,

$$ \log L^C(Y|\theta)= x_A \log(p) + x_B \left[\log (1-p) - \mu \right] + \sum_{k=1}^6 y_k \left[\log(1-p) + k \log(\mu) -\mu - \log(k!) \right] $$

Portanto, o E-Step é dado por,

$$x_{A}^{(t)} = y_0 \frac{p^{(t)}}{p^{(t)} + e^{-\mu ^{(t)}}(1-p^{(t)})} \ \ \text{ e } \ \ x_{B}^{(t)} = y_0 \left[1 - \frac{p^{(t)}}{p^{(t)} + e^{\mu ^{(t)}}(1-p^{(t)})} \right]$$

e o M-Step será dado por,

$$ p^{(t+1)} = \frac{x_{A}^{(t)}}{y_0 + \sum_{k=1}^6 y_k} \ \ \text{ e } \ \ \mu^{(t+1)} = \frac{\sum_{k=1}^6 y_k \times k }{x_B^{(t)} + \sum_{k=1}^6 y_k} $$

Exercício 2 - Ising Model with Periodic External Field

O problema irá definir o grid $\mathbb{Z}^2$, e separará esse grid em dois campos externos diferentes, definidos por dois conjuntos mutuamente exclusivos denominados $Z_-$ e $Z_+$.

  • Os pontos que pertencem ao conjunto $Z_-$ possuem campo de valor $-h_-$
  • Os pontos que pertencem ao conjunto $Z_+$ possuem valor de $h_+$

Um exemplo para ilustrar o problema é dado na figura abaixo, em que $L = 3, l = 1$ (e ignorando $n, m < 1$ e $n, m > 2$).

(a) O algoritmo de Metropolis-Hasting, usando uma função proposta simétrica (lançamento de moeda), é dado por:

  1. Inicializar $\sigma^{0}$

  2. Repetir para $k = 0$ até $N$:

    • Amostrar $(i,j) = (\lfloor u_1 \rfloor$, $\lfloor u_2 \rfloor$), em que $u_1$ ~ $U[mL, (m+1)L - 1]$ e $u_2$ ~ $U[nL, (n+1)L - 1]$

    • Amostrar $p$ ~ Bernoulli$\left(\frac{1}{2}\right)$

    • Amostrar $\sigma^*$, em que $\sigma^*_{(m,n)} =$ $ \begin{cases} \sigma^{k}_{(m,n)},& \text{se } (m,n) \neq (i,j) \\ -\sigma^{k}_{(i,j)},& \text{se } (m,n) = (i,j) \text{ e } p \geq \frac{1}{2}\\ +\sigma^{k}_{(i,j)},& \text{se } (m,n) = (i,j) \text { e } p < \frac{1}{2}\\ \end{cases} $

    • Amostrar $u$ ~ $U[0,1]$

      2.1 Se $u < \rho(\sigma^k, \sigma^*) = \min \left\{1, e^{-\beta \left(H(\sigma^*) - H(\sigma^k)\right) }\right\}$:

                   $\sigma^{k+1} = \sigma^*$,

      Senão,

                   $\sigma^{k+1} = \sigma^{k}$

  3. Fim

Ressaltando que, $$ e^{-\beta \left(H(\sigma^*) - H(\sigma^k)\right) } = \exp\left[-\beta\left(-\sum_{<t,s>} \sigma^*_t \sigma^*_s - \sum_t h_t\sigma^*_t + \sum_{<t,s>} \sigma_s^k\sigma_t^k + \sum_t h_t\sigma^k_t\right)\right] $$

(b) Supondo que tenhamos $\sigma=[\sigma_1, \sigma_2, \sigma_3, \sigma_4] = [\sigma_{(1,1)}, \sigma_{(1,2)},\sigma_{(2,1)}, \sigma_{(2,2)}]$, então pode-se utilizar a amostragem de Gibbs, dada por,

$$\sigma_1|\sigma^{k-1}_2,...,\sigma^{k-1}_M \longrightarrow \sigma_2|\sigma^{k-1}_1,\sigma^{k-1}_3,...,\sigma^{k-1}_M \longrightarrow ........ \longrightarrow \sigma_M|\sigma^{k-1}_1, ..., \sigma^{k-1}_{M-1} \qquad \text{ em que } \sigma_i^k \in \sigma^k \text { onde } i \in \mathbb{Z}^2 \text{ e } k \text{ representa a k-ésima iteração.}$$

Para utilizar a amostragem de Gibbs, podemos começar escrevendo,

$$ P(\sigma) = P(\sigma_1, \sigma_2,\sigma_3,\sigma_4) = \frac{\exp\left[-\beta H(\sigma_1, \sigma_2, \sigma_3, \sigma_4) \right]}{K} $$

E para conseguirmos amostrar $\sigma_1$, (vamos ignorar o sobrescrito $k$, que representa as iterações, para simplificar as contas)

$$ P(\sigma_1 | \sigma_2, \sigma_3, \sigma_4) = \frac{\frac{P(\sigma_1, \sigma_2, \sigma_3, \sigma_3)}{K}}{\frac{P(\sigma_2, \sigma_3, \sigma_4)}{K}} = \frac{P(\sigma_1, \sigma_2, \sigma_3, \sigma_3)}{P(\sigma_2, \sigma_3, \sigma_4)} $$

Podemos então expandir a equação acima, tal que,

$$ P(\sigma_1|\sigma_2, \sigma_3, \sigma_4) = P(\sigma_{(1,1)}|\sigma_{(1,2)}, \sigma_{(2,1)}, \sigma_{(2,2)}) \\ = \frac{ \exp\left[ \beta \left( ( \sigma_{(1,1)}\sigma_{(1,2)} + \sigma_{(1,1)}\sigma_{(2,1)} ) + ( \sigma_{(1,2)}\sigma_{(1,1)} + \sigma_{(1,2)}\sigma_{(2,2)} ) + ( \sigma_{(2,1)}\sigma_{(1,1)} + \sigma_{(2,1)}\sigma_{(2,2)} ) + ( \sigma_{(2,2)}\sigma_{(1,2)} + \sigma_{(2,2)}\sigma_{(2,1)} ) + ( h_{(1,1)}\sigma_{(1,1)} + h_{(1,2)}\sigma_{(1,2)} + h_{(2,1)}\sigma_{(2,1)} + h_{(2,2)}\sigma_{(2,2)}) \right) \right] } { \exp\left[ \beta \left( ( \sigma_{(1,2)}\sigma_{(2,2)} ) + ( \sigma_{(2,2)}\sigma_{(2,1)} + \sigma_{(2,2)}\sigma_{(1,2)} ) + ( \sigma_{(2,1)}\sigma_{(2,2)} ) + ( h_{(1,2)}\sigma_{(1,2)} + h_{(2,1)}\sigma_{(2,1)} + h_{(2,2)}\sigma_{(2,2)}) \right) \right] } $$

E portanto, teremos que,

$$ P(\sigma_1|\sigma_2, \sigma_3, \sigma_4) = \exp\left[ \beta\left( \sigma_{(1,1)}\sigma_{(1,2)} + \sigma_{(1,1)}\sigma_{(2,1)} + \sigma_{(1,2)}\sigma_{(1,1)} + \sigma_{(2,1)}\sigma_{(1,1)} + h_{(1,1)}\sigma_{(1,1)}) \right)\right] = \exp\left[ \beta \sigma_{(1,1)} \left(2\sigma_{(1,2)} + 2\sigma_{(2,1)} + h_{(1,1)}\right) \right] $$

Ou seja,

$$ P(\sigma_1|\sigma_2, \sigma_3, \sigma_4) = \exp\left[\beta \sigma_1 \left( 2\sigma_2 + 2 \sigma_3 + h_1\right)\right] $$

Nota-se que $\sigma_1$ é vizinho apenas de $\sigma_2$ e $\sigma_3$, e que pode se ignorar o campo externo de todos os outros pontos. É claro que pode-se generalizar esse resultado para qualquer $\sigma_i$, e fazemos escrevendo,

$$ P(\sigma_i | \sigma_{\{j \neq i\} \in \Pi}) = \exp\left[\beta \sigma_i \left(2 \partial \sigma_i + h_i\right) \right] $$

Em que $\partial \sigma_i = \{\sigma_j : \langle i,j \rangle \}$, ou seja, a amostragem para $\sigma_i$ depende tão somente de seu próprio campo externo e da soma da energia de seus vizinhos.

O algoritmo, pode ser descrito como:

  1. Inicializar $\sigma^{0}$

  2. Para $k=1$ até N:

    2.1 Para cada ponto $i \in \Pi$:

    • Tomamos $J(\sigma^k_i) = \exp\left[\beta \sigma_i^{(k-1)} \left(2 \partial \sigma_i^{(k-1)} + h_i\right) \right]$

    • Fazemos que $p = \frac{J(\sigma^k_i)}{1+J(\sigma^k_i)}$ (Isto é, o odds da distribuição de Gibbs)

    • $\sigma^k_i = \begin{cases} \sigma^{(k-1)}_i, \text{ se } p \geq 1/2 \\ -\sigma^{(k-1)}_i, \text { se } p < 1/2 \end{cases}$

  3. Fim

(c, d) Aqui neste caso, nossa taxa de aceitação será dada por,

$$ p(\sigma^*, \sigma^k) = \exp\left[ -\frac{\Delta E}{T} \right] \qquad \Delta E = H(\sigma^*) - H(\sigma^k) $$

E para acharmos a configuração $\sigma_{\min}$ de energia mínima, queremos resolver,

$$ \sigma_{\min} = \arg \min _{\sigma} H(\sigma) $$

Para isso, utilizamos o simulated annealing da seguinte forma,

  1. Inicializar $\sigma, T$

  2. Enquanto $T > T_{\min}$:

    • Amostrar $(i,j) = (\lfloor u_1 \rfloor$, $\lfloor u_2 \rfloor$), em que $u_1$ ~ $U[mL, (m+1)L - 1]$ e $u_2$ ~ $U[nL, (n+1)L - 1]$

    • Amostrar $p$ ~ Bernoulli$\left(\frac{1}{2}\right)$

    • Amostrar $\sigma^*$, em que $\sigma^*_{(m,n)} =$ $ \begin{cases} \sigma_{(m,n)},& \text{se } (m,n) \neq (i,j) \\ -\sigma_{(i,j)},& \text{se } (m,n) = (i,j) \text{ e } p \geq \frac{1}{2}\\ +\sigma_{(i,j)},& \text{se } (m,n) = (i,j) \text { e } p < \frac{1}{2}\\ \end{cases} $

    • Amostrar $u$ ~ $U[0,1]$

      2.1. Se $u < \rho(\sigma, \sigma^*) = \min \left\{1, \exp\left[ -\frac{\Delta E}{T} \right] \right\}$:

                   $\sigma = \sigma^*$,

      Senão,

                   $\sigma = \sigma$

      2.2. $T = 5 \sqrt{T}$

  3. Fim

# Criação das funções que serão utilizadas para a simulação

# Inicia o grid
def initial_grid(L):
    grid = np.zeros((L,L))
    spin = [-1,1]
    for i in range(len(grid)):
        for j in range(len(grid[i])):
            u = np.random.randint(0,2)
            grid[i][j] = spin[u]
    return grid

# Função auxiliar para encontrar os vizinhos de um ponto
def find_neighbors(grid, x):
    '''
    params:
        grid = matrix n por n
        x = tupla (i,j)
    Dado um grid, e um ponto x retorna uma lista de tuplas, cada tupla tem as coordenadas do vizinho de x
    
    returns:
        neighbors = [(i-1, j), (i+1, j), (i, j-1), (i, j+1)] # as coordenadas dependem da localização do ponto x
    '''
    neighbors = []
    (i, j) = x
    if (i, j) == (0, 0):
        n1, n2 = (i, j+1), (i+1, j); neighbors = [n1, n2]
    if i != 0 and i != len(grid)-1 and j == 0:
        n1, n2, n3 = (i-1, j), (i, j+1), (i+1, j); neighbors = [n1, n2, n3]
    if i == len(grid)-1 and j == 0:
        n1, n2 = (i, j+1), (i-1, j); neighbors = [n1, n2]
    if i == 0 and j != 0 and j != len(grid[i])-1:
        n1, n2, n3 = (i, j-1), (i+1, j), (i, j+1); neighbors = [n1, n2, n3]
    if i != 0 and i != len(grid) and j != 0 and j != len(grid[i])-1:
        n1, n2, n3, n4 = (i-1, j), (i+1, j), (i, j-1), (i, j+1); neighbors = [n1, n2, n3, n4]
    if i == len(grid)-1 and j != 0 and j != len(grid[i])-1:
        n1, n2, n3 = (i, j-1), (i-1, j), (i, j+1); neighbors = [n1, n2, n3]
    if i == 0 and j == len(grid[i])-1:
        n1, n2 = (i, j-1), (i+1, j); neighbors = [n1, n2]
    if i != 0 and i != len(grid) and j == len(grid[i])-1:
        n1, n2, n3 = (i, j-1), (i+1, j), (i-1, j); neighbors = [n1, n2, n3]
    if i == len(grid)-1 and j == len(grid[i])-1:
        n1, n2 = (i-1, j), (i, j-1); neighbors = [n1, n2]
    return neighbors

# Calcula a diferença de energia $\Delta E$, que é a diferença entre a hamiltoniana proposta e a hamiltoniana do estado atual
def dE(grid_s, grid_k, h):
    hamiltonian = 0
    for i in range(len(grid_s)):
        for j in range(len(grid_s[i])):
            hamiltonian -= h[(i,j)]*(grid_s[i][j] - grid_k[i][j])
            for neighbors in find_neighbors(grid_s, (i,j)):
                (m, n) = neighbors
                hamiltonian -= grid_s[i][j]*grid_s[m][n] - grid_k[i][j]*grid_k[m][n]
    return hamiltonian

# Calcula o campo externo para ser imputado na hamiltoniana
def create_fields(grid, L, l):
    import itertools
    # Criando Z_negativo
    j = 0
    m = []
    for i in range(0, len(grid)):
        j+=1
        if i%10==0:
            j=0
        if j < l:
            m.append(i)
    z_neg = list(itertools.product(m, m))               
    # Criando Z_positivo
    a = [i for i in range(len(grid))]
    Z2 = set(itertools.product(a,a))
    z_pos = Z2.difference(set(z_neg))
    return z_neg, list(z_pos)

# parâmetros inciais
L = 10
l = 5
h_val = 1

import copy
# criando o grid inicial
grid = initial_grid(10*L)

# criando o campo externo
z_neg, z_pos = create_fields(grid, L, l)
h = {}
for zn in z_neg:
    h[zn] = -h_val
j = []
for zp in z_pos:
    h[zp] = (h_val * l**2)/(L**2 - l**2)
    
# inicializando a simulação
grid = np.ones((10*L,10*L))
T = 50000 
num_of_changes = 100
sigmas = []
num_iters = 0

while T > 0.1:    
    # vai diminuindo o número de mudanças dos spins a cada 2 iterações 
    if num_iters%2==0 and num_of_changes > 10:
        num_of_changes-=1
        
    # criam-se duas variáveis aleatorias uniformes no grid para alterar o valor do spin
    grid_s = copy.deepcopy(grid)
    for i in range(int(num_of_changes)):
        u = np.random.randint(0, len(grid), 2)
        grid_s[u[0]][u[1]] = -grid_s[u[0]][u[1]]
        
    # critério de aceitação 
    u = np.random.uniform(0,1)
    delta_e = -dE(grid_s, grid, h)
    if u < min(1, np.exp(delta_e/T)):
        grid = grid_s
        sigmas.append(grid)
        
    # atualização da temperatura
    T = .9*T
    num_iters+=1
